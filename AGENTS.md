# AGENTS.md

This file provides guidance to AI agents when working with code in this repository.

## Project Overview

Personal website/blog built with Hugo static site generator and the PaperMod theme. Content is markdown files in `content/posts/`, configuration is in `hugo.yaml`.

## Development Commands

```sh
# Install/sync Hugo modules (PaperMod theme)
hugo mod tidy

# Start development server (typically http://localhost:1313)
hugo server

# Build static site to public/ directory
hugo

# Create new post (uses archetype template)
hugo new content/posts/my-post-title.md
```

## Architecture

**Static Site Generator**: Hugo v0.148.2 (version pinned in `.hvm`)

- Uses Hugo modules (`go.mod`) to manage the PaperMod theme
- Theme: [hugo-PaperMod](https://github.com/adityatelange/hugo-PaperMod)

**Content Structure**:

- `content/posts/` - Blog posts (markdown files with YAML frontmatter)
- `content/cv/` - CV/resume pages
- `content/photography/` - Photography portfolio
- `archetypes/default.md` - Template for new content with PaperMod-specific frontmatter

**Configuration** (`hugo.yaml`):

- Site metadata, theme configuration, menu structure
- PaperMod parameters: social icons, home page, search (Fuse.js)
- Markdown rendering with Goldmark (allows raw HTML with `unsafe: true`)
- Syntax highlighting with Chroma (pygments classes)

**Output**:

- `public/` - Built static site (gitignored, generated by `hugo`)

## Post Frontmatter

New posts should use the archetype template which includes:

- `date`, `title`, `tags[]`
- PaperMod options: `showToc`, `TocOpen`, `draft`, `hidemeta`, `comments`
- Display options: `ShowReadingTime`, `ShowBreadCrumbs`, `ShowPostNavLinks`
- Optional: `description`, `canonicalURL`, `cover` image

## Key Details

- **Markdown**: Goldmark renderer with raw HTML enabled
- **Syntax highlighting**: Uses CSS classes (not inline styles)
- **Module system**: Theme managed via Hugo modules, not git submodules
- **Hugo version**: Managed by `.hvm` file (v0.148.2)
